Golf Cart Smart Display

PROJECT UNDER DEVELOPMENT AND NOT STABLE

This project is a golf cart display that integrates a GPS, Meshtastic radio, Home Assistant and possibly Bluetooth, ESPNow, ESPMesh, or Wifi (currently undecided).  The display is developed using EEZ Studio Flow and LVGL on a Cheap Yellow Display 2432S028R or variant (also indecided).

* Various smart functions can be performed using the integrated technologies.
* Time/date
* Speed
* Heading
* Weather forecast (next few hours via Meshtastic beacon)
* Messaging - broadcast and private
* Integration with another ESP32 dedicated to Golf Cart sensors and interaction
    * Automatic headlights
    * Accumulated run time for maintenance reminders
    * Fuel monitor
    * Battery voltage monitor
    * Ambient temperature
