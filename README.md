Golf Cart Display (GCD) for The Villages, FL Retirement Community

PROJECT UNDER DEVELOPMENT AND NOT STABLE

This project is a golf cart display that integrates a GPS, Meshtastic radio, Home Assistant and ESPNow.  The display is developed using EEZ Studio Flow and LVGL on a Cheap Yellow Display 2432S028R.

This is one part of a three part Golf Cart computer system:
* Meshtastic Radio (GCM)
* Golf Cart Display computer (GCD - this project)
* Golf Cart Internal computer (GCI)



Various smart functions can be performed using the integrated system.
* GCD + GCM:
  * Graphical user interface
  * Location, speed, heading, time
  * Weather forecast
  * "Now Playing" daily venue/event entertainment schedule for The Villages
  * Messaging - broadcast and private
* GCM + GCD + GCI:
  * Automatic headlights
  * Accumulated run time for maintenance reminders
  * Fuel monitor
  * Battery voltage monitor
  * Ambient temperature
